# Tipulator 💰

A simple tip calculator web app built with HTML, CSS, and JavaScript. Developed as part of a challenge from Code Institute.

## 🚀 Features

* Calculates total bill including tip
* Splits bill per person
* Service rating validation (poor, good, excellent)
* Interactive UI:

  * Hover effects
  * Move interface left/right
  * Close app functionality

## 🧠 Technologies Used

* HTML5
* CSS3
* JavaScript (DOM manipulation & event handling)
